EthOS Node Setup Guide
  1. Clone the EthOS repository:
    git clone https://github.com/ethos-node/ethos
  2. Navigate to the folder:
    cd ethos
  3. Install dependencies:
    npm install
  4. Copy .env config:
    cp .env.example .env
  5. Compile contracts:
    npx hardhat compile
  6. Run local node:
    npx hardhat node
  7. Deploy contract:
    npx hardhat run scripts/deploy.js --network localhost
